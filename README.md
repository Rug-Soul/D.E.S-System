# Rug Soul – DES (Dynamic Estimation System)
**Automate your luxury rug quotations – open source & community-driven.**

## Overview
Dynamic Estimation System Calculator is a professional, SaaS-style tool designed to accurately estimate the value of luxury hand-knotted rugs. 
This tool is ideal for rug manufacturers, sellers, interior designers and collectors who want fast, precise and customizable rug pricing based on material, size and other features.


## Features

Customizable Inputs:
- Rug Material selection
- Knot Density (per square inch)
- Length & Width with unit conversion (cm, inch, feet, meter)

## Advanced Options:
- Irregular or diamond cut adjustments (+3%)
- Optional add-ons for rug features and finishing

## Output & Reporting:
- Price calculation with detailed breakdown
- PDF quotation generation
- CSV import/export of rug price lists

## Client Dashboard (Optional):
- Update company name & logo
- Edit base pricing
